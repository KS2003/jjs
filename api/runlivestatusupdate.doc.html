RunLiveStatusUpdate

Represents Live Status Updates

Some fields can be missing for various reasons, it is normal that particular object will look like {liveScore: null, currentTest: null, finish: false}. Information in all fields except finish can be inaccurate, incorrect or outdated. You can rely on following: if finish is true, final judging results are available
