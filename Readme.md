# JJS
Flexible and fast judging system

## Development
### Prerequisities
- Rust nightly, with cargo and other tools (can be installed through [Rustup](https://rustup.rs))
## Devtool
Usage: 
```bash
cd devtool
cargo run -- # params
```

e.g. to create tgz archive:
```bash
cargo run -- Pkg
```

You can invoke devtool from any place of project
